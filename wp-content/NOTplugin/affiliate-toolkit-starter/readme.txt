== Description ==

Affiliate Toolkit starter lets you easily integrate Amazon products into your WordPress posts. By using our templates, you can present the products in different boxes on your webpage.

For the latest information visit the plugin homepage:
https://www.affiliate-toolkit.com/

The Documentation can be found her (german):
https://docs.affiliate-toolkit.com/

= Features: =
*   Amazon Product Advertising API to retrieve product data
*   Shortcode generator
*   Bestseller lists
*   Search lists
*   Import amazon products
*   Search products from backend
*   Supports HTTPS (ssl)
*   Compatiblity mode for ASA1
*   Supports all Amazon stores activated for the API: Brazil, Canada, China, Germany, Spain, France, India, Italy, Japan, United Kingdom, USA
*   Connect with your tracking-id (100% your affiliate earnings)
*   100% comptability to caching plugins (using cronjob for update product data)
